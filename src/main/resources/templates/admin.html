Upload a .txt file with the following format:

@@Chapter Name starts a new chapter. All following questions belong to it until the next @@ line or end of file.

# lines form the question (the # is stripped).

Multiple consecutive # lines combine into one multi-line question.

A bare # line (nothing after it) embeds a blank line in the question.

Non-#, non-blank lines after a question block are the answer.

Lines starting with @ (single) are treated as regular content lines.

Example:
@@Java Basics
#What is Java?
A compiled and interpreted language.

#Next question
Its answer here

@@Advanced Java
#What is a record?
An immutable data carrier.
